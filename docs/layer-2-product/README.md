# Layer 2 — Product & Integration Specs

**Статус:** CANONICAL  
**Приоритет:** Высокий

---

## Назначение

Layer 2 содержит **спецификации продукта и интеграций** — API, интерфейсы, интеграционные контракты.

Эти документы:
- описывают **пользовательские интерфейсы** и взаимодействия
- определяют **API и форматы данных**
- фиксируют **интеграционные контракты**
- устанавливают **правила нормализации** входных и выходных данных

**Layer 2 должен соответствовать Layer 0 и Layer 1.**
