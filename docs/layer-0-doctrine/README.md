# Layer 0 — Doctrine / Конституция системы

**Статус:** CANONICAL  
**Приоритет:** Высший

---

## Назначение

Layer 0 содержит **фундаментальные принципы, правила и политики системы** — конституцию GenomAI.

Эти документы:
- определяют **архитектурные инварианты** и запреты
- фиксируют **роли компонентов** системы
- устанавливают **non-negotiable правила**
- являются **основой всех решений**

**Layer 0 выше PRD, задач и спринтов.**

## Правила изменения

- Изменения требуют **архитектурного ревью**
- Breaking changes запрещены без явного обоснования
- Все изменения должны быть **совместимы с нижестоящими слоями**
