# BMad Method - Инструкция по установке

## Установка

Команда `npx bmad-method@alpha install` требует интерактивного ввода, поэтому её нужно запускать **вручную в терминале**.

### Шаги:

1. **Откройте терминал** в корне проекта:
   ```bash
   cd "/Users/mosszxc/Documents/Раб файлы/Проэкты/GenomAI"
   ```

2. **Запустите установку:**
   ```bash
   npx bmad-method@alpha install
   ```

3. **При запросе директории установки:**
   - Нажмите Enter (по умолчанию будет использована текущая директория)
   - Или введите `.` для текущей директории

4. **После установки** BMad Method будет доступен через:
   - Агенты в Cursor/Claude
   - Команды `npx bmad-method`

## Использование после установки

### Инициализация проекта:
1. Загрузите **Analyst agent** в Cursor
2. Запустите: `*workflow-init`
3. Выберите подходящий track (вероятно, "📋 BMad Method")

### Для STEP 07 (Outcome Ingestion):
1. Загрузите **PM agent** → Run `*prd`
2. Загрузите **Developer agent** → Помощь в реализации n8n workflow

## Примечание

Пакет `bmad-method@alpha` уже установлен как dev dependency через `npm install`.
Команда `install` создаёт конфигурационные файлы и агентов в директории `.bmad/`.
